    1. 非const引用不能绑定一个右值，例如
        ```
        int& a = 3 + 5;         // 不正确，不能绑定到一个右值上，因为无法确保后期不会修改这个变量，而这个变量却是常量
        const int& a = 3 + 5;   // 正确
        ```
